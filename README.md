# Book Summary Application

This is an AI-powered book info application that:
- generate book summary
- publication date
- author info
By using either wikipedia or

## Installation
##### 1. Clone the repository
```bash
git clone https://github.com/teehere/book-summary-app.git
```

##### 2. Navigate to the project folder
```bash
cd book-summary-application
```

##### 3. Install dependencies
```bash
pip install -r requirements.txt
```

##### 4. Configure API Key
Create a `.env` file:
```
google_key=YOUR_GOOGLE_GEMINI_API_KEY
```

##### 5. Run the application
```bash
streamlit run app.py
```

The application will open in your browser
